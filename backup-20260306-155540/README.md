# 镜头 Prompt 工作台

直接打开 `index.html` 即可使用。

## 已实现
- 批量上传镜头图片，按纵向镜头流展示。
- 每个镜头支持手写 Prompt、AI 生成 Prompt、对话式修改 Prompt。
- 每个镜头保存 Prompt 历史，可回看并恢复到当前版本。
- 每个镜头支持上方插入、下方插入、删除。
- 数据通过 IndexedDB 保存在浏览器本地，刷新后仍保留。

## 使用方式
1. 在页面顶部填入 OpenAI API Key。
2. 如需，调整模型名和全局风格备注。
3. 批量上传镜头图，或先新增空镜头。
4. 在每个镜头里直接写 Prompt，或点击“AI 生成 Prompt”。
5. 在“修改对话”里输入反馈，点击“让 AI 修改 Prompt”。

## 说明
- 这是纯前端网页，不依赖后端服务。
- API Key 保存在当前浏览器本地数据库里，仅供你本机当前浏览器使用。
- 如果浏览器对本地文件页的网络访问有限，建议用任意本地静态服务器打开本项目目录。
