# 视频提示词工作台

一个用于整理镜头图、生成视频提示词、持续修改与收藏归档的前端工具。

## 主要功能

- 批量上传镜头图
- 为每个镜头手写当前 Prompt
- 基于图片生成视频 Prompt
- 针对当前 Prompt 连续提出修改要求，让 AI 迭代
- 保存 `Prompt归档`
- 收藏镜头到收藏夹
- 在收藏夹中添加多个标签、检索、筛选、移回工作台
- 镜头与收藏数据保存在当前浏览器本地

## 当前支持的模型服务商

- `Gemini`
- `OpenAI`

在页面右上角的 `模型与 API 设置` 中可以选择服务商、填写对应的 API Key，并修改模型名。

## 使用方式

1. 打开 `index.html`
2. 点击右上角 `模型与 API 设置`
3. 选择服务商并填写对应 API Key
4. 批量上传镜头图
5. 直接手写 Prompt，或点击 `AI 生成 Prompt`
6. 在 `你的修改要求` 中继续输入修改意见，点击 `让 AI 修改 Prompt`
7. 需要长期保留的镜头可以点击 `收藏此镜头`

## 工作台说明

- `全局风格备注` 会作为所有镜头生成与修改时的公共参考
- `批量生成模式` 可以选择只生成空 Prompt 镜头，或重写全部有图镜头
- `保存当前版本` 会把当前 Prompt 写入 `Prompt归档`
- `复制 Prompt` 只复制当前镜头里的当前 Prompt

## 收藏夹说明

- 收藏的是镜头快照，不会自动跟工作台实时双向同步
- 再次收藏同一个镜头时，会更新该收藏内容
- 标签支持多个
- 已有标签只能通过标签上的 `×` 删除
- 输入框只用于新增标签，不用于删除已有标签

## 数据保存

这个工具当前使用浏览器的 IndexedDB 本地保存：

- 工作台镜头
- Prompt归档
- 修改对话
- 收藏夹
- 标签
- 模型设置
- API Key

这些数据默认只存在当前浏览器环境里。

## 关于 API Key

- API Key 不会自动写进项目目录里的代码文件
- API Key 主要保存在当前浏览器本地数据库中
- 这个版本是前端直连模型服务商接口

因此需要注意：

- 每个使用者都需要填写自己的 API Key
- 浏览器运行时会直接用这个 key 请求模型服务商
