# 视频提示词工作台

一个用于整理镜头图、生成视频 Prompt、持续修改、收藏归档的纯前端工具。

现在同时支持：
- 单镜头 / 图生视频
- 镜头组 / 多图参考
- 收藏夹批量处理
- 多服务商与自定义 API 接入

## 主要功能

- 批量上传镜头图
- 为单镜头生成、修改、归档 Prompt
- 为镜头组基于多张图和镜头顺序生成 Prompt
- 为每个镜头组子镜头填写 `镜头内容`
- 在工作台中收藏镜头，并支持 `覆盖收藏` / `新建收藏`
- 在收藏夹里检索、筛选、打标签、移入工作台
- 收藏夹支持批量选择、批量打标签、批量移入工作台、批量删除
- 收藏夹支持批量导出选中收藏、导入收藏
- 工作台支持导出工作区、导入工作区、导出全部 Prompt
- 所有镜头、收藏、标签、Prompt 归档、对话记录保存在当前浏览器本地

## 当前支持的模型服务商

- `Gemini`
- `OpenAI`
- `自定义 API`

在右上角的 `模型与 API 设置` 中可以：
- 选择 `Gemini` 或 `OpenAI`
- 手动填写模型名
- 填写各自 API Key
- 选择 `自定义 API`
- 手动填写 API 名称、Base URL、模型名、API Key
- 为自定义 API 选择请求模式：
  - `Responses API`
  - `Chat Completions API`

自定义 API 适用于 OpenRouter、SiliconFlow 等兼容 OpenAI 的服务。  
Kimi、MiniMax 这类服务更建议先试 `Chat Completions API`。

## 使用方式

1. 打开 `index.html`
2. 点击右上角 `模型与 API 设置`
3. 选择服务商并填写 API Key / 模型信息
4. 在工作台创建单镜头或镜头组
5. 上传单张图，或为镜头组上传多张图
6. 填写 `镜头标题`、`导演讲戏`，镜头组还可以填写每个子镜头的 `镜头内容`
7. 点击 `AI 生成 Prompt`
8. 如果需要继续调整，在 `你的修改要求` 中输入反馈，点击 `让 AI 修改 Prompt`
9. 需要长期保留时，点击 `收藏此镜头`

## 工作台说明

- `全局提示词风格备注` 会作为所有镜头生成与修改时的公共参考
- `批量生成模式` 可以选择只生成空 Prompt 镜头，或重写全部有图镜头
- 镜头组支持从上方统一上传多图，也支持逐个子镜头替换或删除图片
- `保存当前版本` 会把当前 Prompt 写入 `Prompt归档`
- `复制 Prompt` 只复制当前镜头里的当前 Prompt

## 收藏夹说明

- 收藏的是镜头快照，不会与工作台实时双向自动同步
- 已关联收藏的工作台镜头支持：
  - `覆盖收藏`：更新当前关联收藏
  - `新建收藏`：在保留原收藏的情况下再新建一条
- 镜头组收藏详情中会显示全部子镜头；即使某些子镜头没有图片，也会显示 `镜头内容`
- 批量处理模式下，点击收藏卡片即可选中或取消
- 删除单个收藏或批量删除时，都会先弹出确认提示

## 数据保存

当前使用浏览器的 IndexedDB 本地保存：

- 工作台镜头
- Prompt 归档
- 修改对话
- 收藏夹
- 标签
- 模型设置
- API Key
- 自定义 API 配置

这些数据默认只存在当前浏览器环境里。

## 关于 API Key

- API Key 不会写入项目源码文件
- API Key 主要保存在当前浏览器本地数据库中
- 这个版本是前端直连模型服务商接口

因此需要注意：

- 每个使用者都需要填写自己的 API Key
- 浏览器运行时会直接使用这个 key 请求模型服务商
